# Version (Go)

A tool for bumping up a [semantic version](https://semver.org/).

## Installation

Download the binary.

## Usage

```
NAME:
   version-go bump

USAGE:
   version-go bump [command options] [arguments...]

OPTIONS:
   --write value, -w value  File to write version.
   --read value, -r value   File to output version.
   --help, -h               show help
```
